## Q1

プログラム上に宣言するアクセス修飾子とその説明のくみあわせについて、正しいものを選びなさい。

クラス外まで公開するには [    ]

### 選択肢

#### すべて

- public 修飾子を宣言する
- private 修飾子を宣言する
- protected 修飾子を宣言する
- 修飾子を宣言しない

### ヒント1

クラス内まで公開するには、 private 修飾子を宣言する

### ヒント2

サブクラス内まで公開するには、protected 修飾子を宣言する

### ヒント3


同じパッケージ内まで公開するには、アクセス修飾子を宣言しない

クラス外まで公開するには、 public 修飾子を宣言する


答え

public 修飾子を宣言する

## Q13 入力形式

下のprintメソッドを、クラスの外にも公開したい。空欄に当てはまるアクセス修飾子を記載しなさい。

```java
class Printer {

  [  1  ] void print() {
    // 処理は省略している
  }

}
```

### ヒント1

アクセス修飾子は、フィールドの型宣言の前か、メソッドの返値の前に宣言する。

### ヒント2

-------- フィールドの場合 --------

**アクセス修飾子** 型 変数名;

-------- メソッドの場合 --------

**アクセス修飾子** 戻り値 メソッド名（引数）{

}


### ヒント3

クラス外までフィールドやメソッドを公開することを示すアクセス修飾子は、 public である。


答え

1. public
